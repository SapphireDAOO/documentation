# Introduction

SapphireDAO is an open-source, decentralized payment platform that allows users to pay for goods and services using cryptocurrency. Smart contracts process payments, which are kept in escrow until release requirements are fulfilled. The authorized marketplace wallet handles sensitive actions and dispute resolution.&#x20;

Sellers generate invoices directly in the [Simple Processor](../how-to-guide/markdown.md). In contrast, invoices can also be generated through third-party platforms via the [Advanced Processor](../technical-docs/core-contracts/advancedpaymentprocessor.sol.md), which offers additional features such as dispute resolution, support for payments using fungible tokens, and [meta-invoices](invoices.md#meta-invoice)
